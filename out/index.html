This project is about the Convex Hull problem. It uses two algorithms to solve this problem: the Jarvis March algorithm and the Kirkpatrick-Seidel algorithm. The Jarvis March algorithm, also known as the Gift Wrapping algorithm, is a simple, yet inefficient algorithm. On the other hand, the Kirkpatrick-Seidel algorithm, also known as the Ultimate Planar Convex Hull algorithm, is a more efficient, yet complex algorithm. This project provides a comparison between these two algorithms.
